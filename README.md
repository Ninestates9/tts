# 软件工程课设-tts模型部署  
## 项目名称  
基于VITS的TTS模型移动部署
## 项目目标  
1.熟悉使用Pytorch  
2.了解模型结构和输入输出  
3.Python下使用模型进行推理或预测  
4.对模型进行量化  
5.确定部署方案  
6.转换模型  
7.移动端部署和使用  
## 项目技术栈  
1.训练框架：PyTorch  
2.推理引擎：Onnxruntime  
3.安卓端语言：Java  
4.量化方法：Onnx模型动态量化  
## 参数及onnx格式模型下载地址  
[点击跳转HuggingFace](https://huggingface.co/datasets/Ninestates9/vits_parameter)
